# 자동결제 관리

## 결제 선생 자동 결제

학부모님이 자동 결제 카드를 등록해놓으면 청구서 수령 시 다음 날 오전 10시에 자동 결제 되도록 할 수 있어요.

한도 초과나 카드에 문제가 있어 자동 결제가 진행되지 않은 경우 부모님께 미결제 안내 알림톡이 전달되며 재결제 시도는 진행되지 않아요.
