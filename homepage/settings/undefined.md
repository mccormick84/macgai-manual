# 홈페이지 학원 정보 변경

↖ 상위항목: [홈페이지 세부 설정하기](./)

{% hint style="info" %}
**사용 메뉴**

관리메뉴 → 학원정보 → **학원정보관리**
{% endhint %}

학원정보관리 메뉴에서 홈페이지 하단에 표시되는 정보를 변경할 수 있습니다. 아래의 이미지의 번호에 맞게 정보를 변경하고 <img src="../../.gitbook/assets/btn_저장.png" alt="" data-size="line"> 버튼을 누르면 변경 사항이 적용됩니다.
