# 시험 추가와 관리

{% hint style="success" %}
**이용메뉴**: 기본메뉴 → 수업관리 → **시험관리**
{% endhint %}

1. [시험 생성과 수정](new.md): 새로운 시험을 만들고 저장된 시험을 수정하는 방법을 알아봅니다.&#x20;
2. [시험 종류 관리](category.md): 시험의 영역 정보, 성적표 템플릿 및 누적 관리를 위한 시험 종류 설정 방법을 살펴봅니다.
3. [출력프로그램 (성적표 양식)](undefined.md): 성적표 양식 선택을 위한 출력프로그램 설정 방법을 살펴봅니다.
