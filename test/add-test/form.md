# 출력프로그램 (성적표 양식)

↖ 상위항목: [시험 추가와 관리](./)

출력 프로그램에 따라 성적표의 출력 양식이 달라집니다.

## 영역 명칭 구분

영역 명칭은 수정할 수 없습니다.

* 모의고사 성적표, 모의고사 성적표S, 모의수능 성적표: **영역분류 - 영역 - 유형**
* 입학고사: **행동영역 - 내용영역 - 단원명**
